# Yeti

Yeti is a complete tool set / framework to render the modern Web on all platforms.
Yeti is designed from the ground up with privacy and security in mind.
- Browser Engine
	- DOM
	- CSS
	- V12 - high-performance JavaScript(ECMAScript https://tc39.es/ecma262) and WebAssembly(https://webassembly.github.io/spec/core) engine written in Go

- Browser
